# Battleship!

This is an attempt to complete a simple programming task in a short period of
time. On the east coast the time of the first commit is [[ TIME ]]PM.

## Rules

Make a fully functional battleship library, bonus points for display.

### Gameplay

1. No stacking ships
2. No duplicate shots
3. 15 x 15 board
4. Obviously no diagonal ships
5. No out of bounds shots

### Programmer

There will be two time groupings. The first will be 30 minutes for design and
one hour for programming. There will also be a light rule of no internet aside
from git commits.
